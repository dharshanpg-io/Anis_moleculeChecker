# Chiral Spotlight — R/S Configuration Detector 🧪

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-App-FF4B4B)
![RDKit](https://img.shields.io/badge/RDKit-Cheminformatics-orange)

A beautiful, interactive web application built with **Streamlit** and **RDKit** to deeply analyze the stereochemistry of organic compounds. This app was designed with a premium, glassmorphism-inspired UI to instantly identify chiral carbons and assign precise (R) or (S) configurations.

## ✨ Features
- **Identify Chiral Centers**: Extracts and lists all chiral centers with their correct **(R)** or **(S)** configurations.
- **Visual Mapping**: Highlights chiral atoms automatically on a high-quality 2D structural diagram of the molecule.
- **Comprehensive Counting**: Details total chiral centers as well as Achiral sp³ carbons.
- **1-Click Examples**: Includes instantly loadable examples like **Erythronolide B**, **Artemisinin**, and a complex **Antibiotic**.

## 🚀 Running Locally

1. **Clone the repository:**
   ```bash
   git clone <your-github-repo-url>
   cd <your-repo-folder>
   ```

2. **Install the dependencies:**
   Make sure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```

3. **Start the application:**
   ```bash
   streamlit run streamlit_app.py
   ```
